What is the structure of your dataset?¶

There are 183,412 Bike Trip data in the dataset with details in the columns for:

1- (Duration - start and end time

2- Start and End station and their location in latitude and longitude

3- Bike ID

4- User Type ( Suscriber or one-time Customer)

5- Member Birthyear

6- Member Gender

7- Bike Share for all ( Indicates whether the user is a member of a program for low-income residents)

What is/are the main feature(s) of interest in your dataset?¶

I'm most interested in figuring out what features are best for predicting the ride duration in the dataset.

I think that the strongest effect will be the age of the user in addition to the user type, i am personally interested to check if the gender of the user also affects the bike duration

d- end_station_id Old Dtype float64 Will Convert to Int

e- member_birth_year Old Dtype float64 Will Convert to Int

3- Gender: Will eliminate rows with incomplete data in gender and birthyear details the Other

3- Will create member age from member Birth year

4- Will Eliminate users exceeding 100 years of age as of 2019

6- Gender Will eliminate rows with "Other" listed in Gender

7- Eliminate Rows with no Start or End Stations
